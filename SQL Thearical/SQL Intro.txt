what is SQL?
	- Full form is structured query language.
	- Structured Query Language (SQL) is a standardized programming language that is used to manage relational databases and perform various operations on the data in them
	
Type of SQL commands.
	- There is five type of SQL commands.
		1. DDL
		2. DML
		3. DCL
		4. TCL
		5. DQL
		
	1. DDL(Data Defination languange)
		- DDL changes the structure of the table like creating a table, deleting a table, altering a table, etc.
		- DDL's commands are below 
			- Create 
			- Alter
			- Drop
			- Truncate
		
	2. DML(Data Manipulation Language)
		- DML commands are used to modify the database. 
		- DML's commands are below
			- Insert
			- Update
			- Delete
			
	3. DCL(Data Control Language)
		- DCL commands are used to grant and take back authority from any database user.
		- DCL's commands are below
			- Grant
			- Revoke
			
	4. TCL(Transaction Control Language)
		- There commands are below
			- Commit
			- Rollback
			- Savepoint
			
	5. DQL(Data Query Language)
		- DQL is used to fetch the data from the database
		- DQL's commands are below
			- Select
